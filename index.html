Agricultural Automation

IoT Based Soil Nutrient, Temp., Moisture, pH Monitoring & Management System

Soil is the base of agriculture. Soil provides nutrients that increase the growth of a crop. Some chemical and physical properties of soil, such as its moisture, temperature, soil nitrogen, phosphorous & potassium content heavily affect the yield of a crop. These properties can be sensed by the use of hardware sensors, whose data can be used to monitor and analyse vital growth parameters that can help for better, effective and efficient crop productions.

In this project, IoT based soil profile monitoring & management system is proposed in which the farmer will be able to monitor soil moisture, air temperature, humidity, solar intensity, soil nutrient contents like Nitrogen, Phosphorous & Potassium along with soil pH. The farmer can monitor all these parameters wirelessly on a mobile phone or the PC.

With the use of this system, field irrigation can be automated on the basis of dryness of soil. Various alert system can also be induced, based on the requirements of the client. Apart from that air temperature, humidity and available sunlight (vital parameters for disease control) can be tracked using our deployed sensors from the field continuously, thanks to our robust technology enabling seamless data transmission from the sensors.

System will use a server to monitor the data in graphical & numerical format. There is a need of a GSM or Wi-Fi network in the field to send the data to the server. But the agricultural field doesn’t have access to these networks. In order to solve this, a wireless transceiver is used to send the data from sensor node to gateway. The data from the transmitter can be transmitted wirelessly from a kilometre distance to the receiver. The receiver is built using Wi-Fi enabled MCU, which has access to Wi-Fi Network. Using this Wi-Fi Network, the data can be uploaded to the server.
